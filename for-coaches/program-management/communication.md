## Communication Protocols

<div class="accordion-section">
  <button class="accordion-toggle">General Communication Guidelines</button>
  <div class="accordion-content">
    <div class="accordion-content-inner">
      <h3>Timeline for Regular Communications</h3>
      <ul>
        <li><strong>Registration confirmation:</strong> Within 24 hours of registration</li>
        <li><strong>Welcome email:</strong> 7 days before program start</li>
        <li><strong>Final reminder:</strong> 2 days before program start</li>
        <li><strong>Session follow-up emails:</strong> Within 24 hours after each session</li>
        <li><strong>Next session reminder:</strong> 1-2 days before each session</li>
        <li><strong>Program completion email:</strong> Within 3 days of final session</li>
        <li><strong>Follow-up on next steps:</strong> 14 days after program completion</li>
      </ul>
      
      <h3>Email Best Practices</h3>
      <ul>
        <li>Always use the program email account for all communications</li>
        <li>CC the program coordinator on all group communications</li>
        <li>Use consistent branding and formatting in all emails</li>
        <li>Include contact information in all communications</li>
        <li>Respond to participant inquiries within 24 hours</li>
        <li>Maintain a professional tone while being encouraging and supportive</li>
        <li>Use BCC when sending to multiple participants (except for program coordinators)</li>
      </ul>
      
      <h3>Emergency Communication Chain</h3>
      <ol>
        <li>Primary Coach notifies Program Coordinator immediately</li>
        <li>Program Coordinator contacts affected participants via call/text</li>
        <li>Follow-up email sent with all details once situation is resolved</li>
        <li>Incident report filed within 24 hours with communications log</li>
      </ol>
    </div>
  </div>
</div>

<script>
  function copyToClipboard(id) {
    const element = document.getElementById(id);
    if (!element) return;
    
    const text = element.innerText;
    navigator.clipboard.writeText(text).then(() => {
      const button = event.target;
      const originalText = button.innerText;
      button.innerText = 'Copied!';
      setTimeout(() => {
        button.innerText = originalText;
      }, 2000);
    });
  }
</script>
