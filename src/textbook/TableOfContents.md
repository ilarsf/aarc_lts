# AARC Learn to Scull Program Textbook

Welcome to the comprehensive textbook for the Ann Arbor Rowing Club's Learn to Scull program. This textbook has been carefully structured to guide both adult learners and volunteer coaches on a clear, progressive learning journey. It aims to transform the program's web-based resources into a cohesive, standalone guide.
