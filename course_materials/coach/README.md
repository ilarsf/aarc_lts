# Coach Materials Organization

## Directory Structure Notice

> **IMPORTANT**: The majority of coach materials have been migrated to the password-protected `/coach_portal/` directory.

This directory (`/course_materials/coach/`) is being maintained for legacy purposes, but all active content development and updates should be made in the new Coach Portal structure.

## Material Locations

- **Coach's Manual**: `/coach_portal/manual/Coach_Manual.md`
- **Daily Coach Checklist**: Available as a printable PDF in `/assets/pdf/daily_coach_checklist.html`
- **Participant Assessment Checklist**: `/coach_portal/assessment/Participant_Assessment_Checklist.md`
- **Session Plans**: `/coach_portal/session_plans/`
- **Technical Resources**: `/coach_portal/technical/`
- **Safety Procedures**: `/coach_portal/safety/`

## File Synchronization

If you're updating content, please make sure to update it in the Coach Portal locations rather than in this legacy directory structure.

If you find any broken links or navigation issues, please report them to the site administrator.
